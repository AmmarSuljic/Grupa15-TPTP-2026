# Grupa15-TPTP-2026 Audi BiH stranica

## O projektu
Web stranica Audi BiH prikazuje različite modele vozila i omogućava interakciju kroz funkcionalnosti kao što su filtriranje vozila, kontakt forma sa validacijom, 
leasing kalkulator, statistika vozila, newsletter prijava i cookie banner. Projekat je realizovan korištenjem HTML5, CSS3 i JavaScript tehnologija,
sa fokusom na responzivan dizajn, korisničko iskustvo i dinamičko upravljanje sadržajem.
